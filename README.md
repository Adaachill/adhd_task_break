# タスクブレーキ（仮称）

**「やる気ゼロ」と「やりすぎ」を同時に解決する、ADHD特化型行動コントロールアプリ**

LINEの手軽さでタスクを吐き出し、松竹梅で着手のハードルを下げ、強制ブレーキタイマーで過集中を止める。

---

## コンセプト

一般的なタスク管理アプリは「サボり防止」に偏りがちですが、ADHDには「着手できない」だけでなく「やりすぎてしまう（過集中）」という痛みもあります。タスクブレーキはこの2つを同時に解決します。

**こんな悩みに刺さります：**
- タスク管理ツールが続かない
- メモが溜まりすぎて優先順位を見失う
- 着手するまでに莫大な時間がかかる
- 集中すると時間を忘れて数時間やり続けてしまう

---

## 主な機能（MVP）

### ① 脳内吐き出し（チャットUI）
LINE風のチャットバブルでタスクを摩擦ゼロで入力。送信と同時にAIが🔵（通常）/🔥（緊急）と期限を自動判定。

### ② 松竹梅クリアシステム
🔵タスクには「松・竹・梅」の3段階の達成基準を設定。「教科書を1ページ開いたら梅クリア！」のように、意識の低さを全肯定します。

### ③ 強制ブレーキタイマー
🔥タスクには制限時間を設定（既定45分）。時間になると強制アラートが発火し、「15分延長」か「強制終了して休憩」の2択を迫ります。バックグラウンドやアプリ終了後も確実に発火。

### ④ 時間計測 / 単発・習慣切替
作業時間を自動計測。単発タスクと繰り返し習慣タスクを切り替えて管理できます。

### ⑤ 褒めログ
完了したタスクだけを表示。未完了は非表示にして自己嫌悪を防ぎます。「今日 〇〇分 動けた！」を可視化し、SNSシェアにも対応。

### ⑥ 作業中離脱抑止
作業セッション中に他アプリへ移ると「〇〇の作業中です。関連ありますか？」と引き戻し通知。常時バナーで今やっていることを画面に表示し続けます。

## 技術スタック

| カテゴリ | 技術 |
|---|---|
| フロントエンド | React Native + Expo (TypeScript) |
| ビルド | EAS Build |
| 状態管理 | Zustand または hooks + ローカルDB |
| データ永続化 | expo-sqlite（オフラインファースト） |
| 通知・タイマー | expo-notifications / expo-haptics |
| 常時バナー | Android: フォアグラウンドサービス / iOS: Live Activity |
| AI自動分類 | FastAPI + Claude API（プロンプトキャッシュで最適化） |

**オフラインファースト** — タスクデータはデバイス内に完結。ADHDのタスクは機微情報のため、プライバシー的にも有利。
